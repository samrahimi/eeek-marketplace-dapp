Setting up Dev Tools

Unzip the theme package to any folder and open a command line at that location.

Wingman's dev tools require Node, Git, and the Gulp CLI package. Assuming you have Node and Git installed, the following command installs Gulp Command Line Utility globally so the gulp command can be used anywhere in your system.

npm install gulp-cli -g

When you have installed Gulp, run npm install and the required packages for the Wingman dev tools will be downloaded to the node_modules directory.

npm install

Wingman employs Browsersync to serve pages from the dist directory. Running gulp will compile the theme, copy all required files to the dist directory and will open a browser window to dist/index.html.

gulp

While Gulp is running, Files in the pages, scss and js folders are monitored for changes, which will inject updated CSS or cause a refresh in Browsersync.

Hit Ctrl+C or just close the command line window to stop the server.
